Prework Study Guide

HTML

  • Used to specify structure of visual elements.
  • The head element contains information about the webpage.
  • Most important heading content should be the h1 tag.
  • The body element represents the visible content shown to the user.
  • Uses parent and child hierarchy elements to identify relationship.
  • Must have an opening and closing tag.
  • Alt attribute contains textual description of image, which is important to make page accessible to visually impaired users that may use screen readers.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch name: creates a new branch and switches to it.
  • add -A: commit all files in a working branch.
  • Version control system that helps you to collaborate with other developers.
  • Committing to a Git repository allows you to keep track of all the changes made since the last file was committed.
